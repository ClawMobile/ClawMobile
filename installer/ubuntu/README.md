# Legacy Full Backend Ubuntu Scripts

This directory belongs to the advanced DroidRun/MobileRun backend. It is used by
the legacy full installer to prepare Ubuntu/proot-side dependencies.

Most users should start from the default ClawMobile install in
[../INSTALL.md](../INSTALL.md). Use this directory only when working on the full
DroidRun/MobileRun backend described in [../FULL_BACKEND.md](../FULL_BACKEND.md).

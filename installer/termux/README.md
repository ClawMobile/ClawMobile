# Legacy Full Backend Termux Scripts

This directory belongs to the advanced DroidRun/MobileRun backend.

Most users should install ClawMobile through the default Termux runtime:

```sh
../termux-lite/clawmobile setup --quick
clawmobile run
```

Use these scripts only when you intentionally want the legacy full backend with
Ubuntu/proot, DroidRun Portal, and DroidRun/MobileRun agent execution.

See [../FULL_BACKEND.md](../FULL_BACKEND.md) for details.

# Security & Privacy

Clawbot Mobile runs entirely on the local Android device.

## Data handling

- No screen streaming is performed
- No UI data is uploaded by default
- API keys are provided by the user via environment variables
- No credentials are committed to this repository

## Android permissions

DroidRun requires:
- ADB debugging
- Accessibility service

These permissions are required for UI automation.

## Responsible use

This project controls the local device.

Users are responsible for:
- Reviewing permissions
- Understanding automation behavior
- Complying with app and system policies
